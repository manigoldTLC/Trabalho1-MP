No primeiro caso de teste é verificado todos os erros possiveis, retornando -2 caso o jogo
seja impossível pelas regras, depois é testado se o jogo não foi iniciado, retornando -1
caso todos os campos estejam vazio.

No segundos caso de teste é verificado todas as situações que podem ocorrer empate
retornando 0 caso tenha acontecido todas as jogadas.

No terceiro caso de teste, é verificado todos os casos em que o X ganha pelas linhas,
retornando 1 caso seja verdade.

Similarmente no quarto caso de teste é verificado todos os casos em que O ganha pelas linhas
retornando 2 caso seja verdade.

No quinto caso de teste, é verificado todos os casos em que o X ganha pelas colubas,
retornando 1 caso seja verdade.

Similarmente no sexto caso de teste é verificado todos os casos em que O ganha pelas colunas
retornando 2 caso seja verdade.

No sétimo caso de teste é verificado todos os casos em que o X ganha pelas diagonais, retornando
1 caso verdadeiro

No oitavo caso é verificado todos os casos em que O ganha pelas diagonais
retornando 2 caso verdadeiro.

É feito mais um caso de teste, sendo esse testes extras para verificar se o código está funcionando
plenamente.
